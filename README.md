

Dette projekt er en landing page for den fiktive højtaler Lumina One. Formålet med projektet er at præsentere produktet på en visuel og brugervenlig måde, hvor brugeren nemt kan få information om produktet, se forskellige farver og få lyst til at købe Lumina One.

Projektet er lavet med HTML, CSS og JavaScript. HTML er brugt til at strukturere indholdet på siden, CSS er brugt til styling, layout, farver og visuelle effekter, mens JavaScript er brugt til interaktive funktioner som farveskift og åbning/lukning af produktspecifikationer.

På siden kan brugeren blandt andet se produktbilleder, vælge forskellige farvevarianter, læse produktspecifikationer, se stemningsbilleder og læse anmeldelser. Projektet arbejder primært med produktdata som navn, pris, farver, billeder, specifikationer og anmeldelser.



Mappestruktur 
Projektet er organiseret i forskellige filer og mapper for at gøre det mere overskueligt at arbejde med.

Min index.html-fil indeholder selve strukturen på landing pagen. Her har jeg opbygget sidens forskellige sektioner, såsom header, hero-sektion, produktdetaljer, produktspecifikationer, stemningsbilleder, farvevalg, anmeldelser og footer.

Mappen css indeholder min CSS-fil, landingpage.css. Den bruges til at style hele siden, blandt andet layout, farver, typografi, knapper og billeder.

Mappen js indeholder min JavaScript-fil, landingpage.js. Den bruges til sidens interaktive funktioner, for eksempel farveskift på produktet og åbning/lukning af produktspecifikationer.

Mappen img indeholder alle billeder og ikoner, der bruges på siden. Her ligger blandt andet produktbilleder af højtaleren i forskellige farver, stemningsbilleder samt ikoner som søgning, kurv og hjerte.

Jeg har valgt denne struktur, fordi den gør projektet mere ryddeligt og nemmere at finde rundt i. Ved at opdele HTML, CSS, JavaScript og billeder i separate mapper bliver koden mere overskuelig, og det er lettere at rette eller videreudvikle projektet.
